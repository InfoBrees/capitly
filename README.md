# Capitly 

**Capitly** is a modern finance overview app that helps you track, analyze, and manage your personal finances — all in one place.

It combines the flexibility of spreadsheets with a clean, user-friendly interface, giving you full control over your financial data without the complexity of traditional tools.

---

## Features

### Spreadsheet-like Calculations

* Use familiar, Excel-style calculations directly in the app
* Create custom formulas to track budgets, expenses, and income
* Flexible structure — adapt it to your personal workflow

---

### CSV Import (Excel / Numbers)

* Easily import existing data from:

  * Excel (.csv)
  * Apple Numbers (.csv)
* Seamless migration from your current spreadsheets
* No need to start from scratch

---

### Financial Overview

* Get a clear overview of:

  * Income
  * Expenses
  * Savings
* Organize your financial data in a structured and visual way

---

### Built for Simplicity

* Clean and intuitive interface
* No steep learning curve
* Designed for everyday use

---

## Use Cases

* Personal finance tracking
* Budget planning
* Expense monitoring
* Migrating from Excel/Numbers to a dedicated app
* Custom financial calculations

---

## Tech Stack

* **Backend:** Java + Spring (Maven)
* **Frontend:** Svelte + TypeScript
* **Data Handling:** CSV import/export

---

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/capitly.git
cd capitly
```

### 2. Run backend

```bash
cd backend
mvn spring-boot:run
```

### 3. Run frontend

```bash
cd frontend
npm install
npm run dev
```
